# 🧠 Interactive Quiz Application

A modern and responsive Quiz Application developed using **HTML**, **CSS**, and **JavaScript**. This project tests users with multiple-choice questions, instantly displays correct and incorrect answers, keeps track of the score, and allows users to restart the quiz after completion.

---

## 🚀 Features

- 🎯 Multiple Choice Questions
- 🔀 Randomized Question Order
- ✅ Instant Answer Validation
- 📊 Live Score Tracking
- 🎨 Responsive & Colorful UI
- 🔄 Restart Quiz Option
- ⚡ Fast and Lightweight
- 💻 Beginner-Friendly Source Code

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/yourusername/Quiz-Application.git
```

2. Open the project folder.

3. Open **index.html** in your web browser.

No installation or additional software is required.

---

## 🎮 How to Play

- Click the **Start** button.
- Read the question carefully.
- Select the correct answer.
- Click **Next** to continue.
- View your final score at the end.
- Press **Restart** to play again.
